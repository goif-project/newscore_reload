### この開発環境について  
react + pug + stylus + webpack + ES6

##### 開発を始める前に...
```
npm i             // ダウンロードしたらまずこのコマンド
npm run start     // 開発用サーバ(localhost:3000)を立ち上げる
npm run build     // js、stylusファイルをbundle.jsにコンパイル
```

---

##### 各フォルダについて  
* dist  
  コンパイル後の本番up用ファイルの出力先  
  この中のファイルは編集しないこと！

* src
  各種コンパイル前コードの格納場所
